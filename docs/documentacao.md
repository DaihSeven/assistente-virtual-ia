# HealthAssist AI
## Descrição

O HealthAssist AI é um assistente virtual baseado em Inteligência Artificial criado para auxiliar usuários com informações básicas relacionadas à saúde preventiva e hábitos saudáveis.

O *objetivo* é facilitar o acesso a informações confiáveis, ajudando usuários a compreenderem temas de saúde de forma simples e organizada.

## Público-alvo

Pessoas que desejam obter informações iniciais sobre:

alimentação saudável;
exercícios físicos;
prevenção de doenças;
hábitos de qualidade de vida.
Funcionalidades

## O assistente consegue:

✔ responder dúvidas utilizando uma base de conhecimento;

✔ explicar conceitos de saúde;

✔ sugerir hábitos saudáveis;

✔ indicar quando uma informação está fora do seu conhecimento;

✔ orientar o usuário a buscar um profissional quando necessário.

## Limitações

O assistente:

não realiza diagnósticos;
não substitui consultas médicas;
não recomenda medicamentos;
não deve criar informações que não estejam na base de conhecimento.
