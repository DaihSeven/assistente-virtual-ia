# 🎤 Pitch: HealthAssist AI
## Problema

Muitas pessoas possuem dúvidas relacionadas à saúde e qualidade de vida, mas nem sempre encontram informações organizadas, simples e confiáveis de forma rápida. Além disso, informações incorretas podem levar a interpretações equivocadas e decisões inadequadas.

## Solução

O HealthAssist AI é um assistente virtual baseado em Inteligência Artificial criado para auxiliar usuários com informações sobre saúde preventiva e hábitos saudáveis.

A solução utiliza uma base de conhecimento estruturada para responder perguntas sobre temas como alimentação, exercícios físicos, hidratação, sono e bem-estar, sempre buscando fornecer respostas claras e evitar informações sem fundamento.

## Como funciona

O assistente:

recebe uma dúvida do usuário;
analisa a pergunta;
consulta uma base de conhecimento organizada;
retorna uma resposta relacionada ao tema;
informa quando não possui informações suficientes.
## Tecnologias utilizadas
Python: desenvolvimento da aplicação;

Streamlit: criação da interface interativa;

JSON: armazenamento da base de conhecimento;

Inteligência Artificial Generativa: conceito aplicado ao comportamento do assistente;

Processamento de Linguagem Natural (PLN): interação com perguntas em linguagem humana.
## Diferencial

O HealthAssist AI foi desenvolvido com foco em respostas responsáveis, evitando criar informações falsas e deixando claro seus limites.

O projeto demonstra como a Inteligência Artificial pode ser aplicada para facilitar o acesso à informação e melhorar a experiência do usuário.

## Possíveis próximos passos

Como evolução futura, o projeto pode receber:

integração com modelos de IA generativa reais;

busca semântica utilizando embeddings;

banco de dados para histórico de conversas;

autenticação de usuários;

arquitetura com API e frontend separados.

# Em resumo:

"O HealthAssist AI é um assistente virtual desenvolvido em Python que utiliza uma base de conhecimento para responder dúvidas sobre saúde preventiva. O objetivo do projeto é demonstrar como a Inteligência Artificial pode auxiliar usuários no acesso rápido a informações organizadas, mantendo respostas responsáveis e evitando informações inventadas. A aplicação foi construída utilizando Python, Streamlit e uma estrutura de dados em JSON, com possibilidade de evolução para uma solução baseada em modelos generativos e RAG."
